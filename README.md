# Vehical_managment_system.Vehicle Rental SystemThe Vehicle Rental System is a Java-based application designed to manage the rental operations of different types of vehicles. Built using core Object-Oriented Programming (OOP) principles, the system allows users to calculate rental costs and manage specifications for various vehicle categories including Cars, Bikes, and Trucks. It utilizes an abstract base class to enforce consistency and encapsulate common vehicle behaviors.Key Features:Interactive Vehicle Selection Panel: This text-based interface prompts the user to select their desired vehicle type (Car, Bike, or Truck). Based on the choice, the application handles data capture and processes information specific to that vehicle class.Car Rental Management: This panel handles car-specific records, including unique features like the number of doors. It collects the license plate number and daily rental rates from user input, automatically verifying availability status and calculating the multi-day base rental fees.Bike Rental Management: This component manages motorcycle rentals with custom parameters, such as tracking whether a safety helmet is included. If a helmet is included, the system automatically appends a flat accessory fee ($\$50$) to the standard daily rate calculation.Truck Rental Management: Designed to process heavy transport vehicles, this feature logs the maximum load capacity of the truck. The rental calculation dynamically scales based on the vehicle's capacity, adding a cargo fee proportional to the load capacity (calculated as $\text{Load Capacity} \times 10$) alongside the baseline daily rate.
